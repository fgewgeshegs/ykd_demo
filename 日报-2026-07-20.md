# 日报 2026-07-20

## 完成

### 1. 多轮对话上下文
- 新增 `context/` 包：`ContextStore` 接口 + `InMemoryContextStore` 内存实现，单用户保留 50 条历史
- `ChatService`/`LLMClient` 调用前取历史、调用后写回，支持多轮记忆
- `VisionHandler` 图片分析结果自动写入上下文

### 2. 图片上下文存储
- `ContextStore` 新增 `setLastImage/setLastImageUrl` 存图片 CDN 参数和 AI URL
- `VisionHandler` 四级下载降级：直接 CDN → 上下文 CDN → 上下文 URL → 消息 URL
- `ImageGenerationHandler` 生成后自动存 AI URL，解决 bot 图片 CDN 下载失败问题

### 3. 语音基础设施（队友接入准备）
- `WechatMessage` 新增 `voiceEncryptQueryParam/voiceAesKey/voiceText`
- `ContextStore` 新增 `setLastVoice/getLastVoice`（存 MP3 路径+转写文字）
- `WechatMessageService` pollLoop 加 VOICE 分支

### 4. SDK 升级 1.0.1 → 2.3.3
- 重写 `WechatILinkClient`：builder 模式、executeLogin、黑盒 sendImage/sendVoice
- 重写 `WechatMessageService`：新 API 无 cursor、无 contextToken
- 新增 `sendVoiceMessage` 方法
- 登录二维码兼容 URL 和 base64

### 5. 清理
- 移除引用消息功能（SDK 1.0.1 text=null bug，2.3.3 无此接口）
- 移除图文合并逻辑

## 未完成/待办
- 引用消息：SDK 层面不支持，需等协议更新
- 语音：队友开发 VoiceHandler + ASR + TTS
