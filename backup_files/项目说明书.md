# Claw Assistant 项目说明书

## 1. 项目概述

Claw Assistant 是一个运行在微信平台上的 AI 多模态助手机器人。用户通过微信发送文字或图片，机器人自动调用 LLM/视觉/生图模型完成对话、图片分析和图片生成，并以文字或图片消息形式回复。

**核心技术栈：** Java 21 / Spring Boot 3.3 / 微信 iLink SDK / 阿里云百炼 / DeepSeek

## 3. 架构分层

```
┌──────────────────────────────────────────┐
│              微信 iLink SDK               │  ← 腾讯官方 API
├──────────────────────────────────────────┤
│         WechatMessageService             │  ← 消息轮询、收发
│         MessageRouter                    │  ← 意图路由
│  ┌──────────┬──────────┬──────────────┐  │
│  │ Chat     │ Vision   │ ImageGen     │  │  ← Handler 层
│  │ Handler  │ Handler  │ Handler      │  │
│  └────┬─────┴────┬─────┴──────┬───────┘  │
├───────┼──────────┼────────────┼──────────┤
│       │    AI 能力层              │        │
│  ┌────┴──┐ ┌─────┴────┐ ┌──────┴──────┐ │
│  │Chat   │ │Vision    │ │ImageGen     │ │  ← Service 层
│  │Service│ │Service   │ │Service      │ │
│  └───┬───┘ └────┬─────┘ └──────┬──────┘ │
│  ┌───┴───┐ ┌────┴─────┐ ┌──────┴──────┐ │
│  │LLM    │ │Vision    │ │Image        │ │  ← Client 层
│  │Client │ │Client    │ │Client       │ │
│  └───┬───┘ └────┬─────┘ └──────┬──────┘ │
├──────┼──────────┼────────────┼──────────┤
│  DeepSeek   阿里百炼    阿里百炼           │  ← 外部 AI 服务
│             qwen-vl    qwen-image         │
└──────────────────────────────────────────┘
```

**各层职责：**
- **Client 层** — 封装 HTTP 调用、请求构建、响应解析，与外部 AI API 通信
- **Service 层** — 业务逻辑编排，调用 Client 并处理异常降级
- **Handler 层** — 微信消息协议适配，接收 WechatMessage / 返回 ReplyMessage
- **Router 层** — 意图分类 + 路由分发，不包含业务逻辑

## 4. 核心流程

### 4.1 消息处理主流程

```
微信消息到达
  → WechatMessageService.pollLoop() 轮询获取
  → MessageRouter.route()
      ├─ 图片消息 → VisionHandler → 下载 CDN 图片 → VisionService 分析 → 返回文本
      └─ 文字消息 → IntentClassifier.classify()
          ├─ CHAT           → AIChatHandler → ChatService → LLM 对话
          ├─ IMAGE_GENERATE → ImageGenerationHandler → 生成→下载→上传 CDN→发图片
          └─ IMAGE_ANALYZE  → VisionHandler
  → WechatMessageService 根据 ReplyMessage 类型发送文本或图片
```

### 4.2 图片生成 CDN 上传流程

```
用户: "画一只猫"
  → IntentClassifier → IMAGE_GENERATE
  → ImageGenerationHandler:
      1. ImageClient.generateImage() → 图片 URL
      2. HttpClient GET 下载图片字节
      3. WechatILinkClient.uploadMedia(1, userId, bytes) → CDN 上传
         (AES-ECB 加密 → getUploadUrl → CDN 直传)
      4. 返回 ReplyMessage.image(encryptQueryParam, aesKey, fileSize)
  → WechatMessageService.sendImageMessage(fromUserId, contextToken, mediaInfo)
  → 用户直接看到图片 ✓
```

## 5. 配置说明

配置文件：`src/main/resources/application.properties`

| 配置项 | 说明 | 示例值 |
|---|---|---|
| `wechat.ilink.enabled` | 是否启用微信 iLink | `true` |
| `wechat.ilink.poll-interval-ms` | 消息轮询间隔（毫秒） | `3000` |
| `llm.api-key` | 文本对话 LLM API Key | DeepSeek key |
| `llm.base-url` | LLM API 地址 | `https://api.deepseek.com` |
| `llm.model` | LLM 模型名 | `deepseek-v4-flash` |
| `vision.api-key` | 视觉模型 API Key | 百炼 key |
| `vision.base-url` | 视觉 API 地址 | `https://dashscope.aliyuncs.com/compatible-mode/v1` |
| `vision.model` | 视觉模型名 | `qwen-vl-max` |
| `image.api-key` | 生图模型 API Key | 百炼 key |
| `image.model` | 生图模型名 | `qwen-image-2.0` |
| `image.base-url` | 生图 API 地址 | `https://dashscope.aliyuncs.com/api/v1/services/aigc/multimodal-generation/generation` |
| `weather.api.url` | 天气 API | OpenWeatherMap |
| `weather.api.key` | 天气 API Key | — |

## 6. 启动方式

```bash
# 1. 复制配置模板并填入 API Key
cp src/main/resources/application.properties.example \
   src/main/resources/application.properties

# 2. 编译
mvn clean compile

# 3. 启动（建议指定 UTF-8 编码避免中文乱码）
mvn spring-boot:run -Dfile.encoding=UTF-8

# 或打包运行
mvn clean package -DskipTests
java -Dfile.encoding=UTF-8 -jar target/claw-assistant-1.0-SNAPSHOT.jar
```

启动后控制台会打印微信扫码登录二维码 URL，扫码成功后自动开始监听消息。
