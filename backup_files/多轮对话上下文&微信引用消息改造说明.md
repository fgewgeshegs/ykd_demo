# 多轮对话上下文 & 微信引用消息改造说明

> 日期：2026-07-20  
> 分支：`cxx`（基于 `feature/image-generate` 最新代码）

---

## 一、背景

1. 从 `feature/image-generate` 拉取最新 4 个 commit，fast-forward 合并到 `cxx`。
2. 在合并后的代码基础上，实现多轮对话上下文记忆 + 微信引用消息功能。

---

## 二、改动清单

### 2.1 新增 `context/` 会话上下文包（3 个新文件）

| 文件 | 说明 |
|---|---|
| `context/Message.java` | 单条对话记录 `record(String role, String content)`，role 为 `user` / `assistant` |
| `context/ContextStore.java` | 接口，定义 `getHistory()` / `append()` / `clear()` 三个方法 |
| `context/InMemoryContextStore.java` | 内存实现，`ConcurrentHashMap<userId, Deque<Message>>`，单用户上限 50 条，线程安全 |

### 2.2 AI 层改动

| 文件 | 改动 |
|---|---|
| `LLMClient.java` | 新增 `chat(userId, text, history)` 重载；`buildRequestBody()` 在 system prompt 之后、当前消息之前插入历史消息数组；原 `chat(userId, text)` 委托到新方法传 `List.of()` |
| `ChatService.java` | 注入 `ContextStore`；调用 LLM 前取最近 20 条历史 → 调用后将本轮 user/assistant 消息写回上下文 |

### 2.3 微信层改动

| 文件 | 改动 |
|---|---|
| `WechatMessage.java` | 新增 `refMessageText`、`refImageEncryptParam`、`refImageAesKey`、`hasAttachedImage` 四个字段 |
| `WechatMessageService.java` | 新增 `extractRefMessage()` 提取引用消息的文字/图片/语音 CDN 参数；新增 `buildImageMessage()` / `fillImageFields()` / `routeAndReply()` 辅助方法；`pollLoop()` 改为 indexed for 循环，支持 `IMAGE+TEXT` / `TEXT+IMAGE` 图文合并 |
| `MessageRouter.java` | 图片消息当 `hasAttachedImage=true` 时先走意图分类；`fallbackIfEmpty()` 基于 `WechatReply.hasContent()` 判断 |
| `AIChatHandler.java` | 新增 `buildFullText()`，将引用文字拼成 `[用户引用了: "xxx"]\n用户消息` 传给 LLM |
| `VisionHandler.java` | 注入 `ContextStore`，图片分析完成后将 `[用户发送了一张图片]` + AI 分析结果写入上下文 |

## 三、核心流程

### 3.1 多轮对话

```
用户发 "你好"
  → ChatService.chat(userId, "你好")
    → contextStore.getHistory(userId, 20)  // []
    → llmClient.chat(userId, "你好", [])
    → contextStore.append(userId, "user", "你好")
    → contextStore.append(userId, "assistant", "你好！有什么可以帮你的？")

用户接着发 "刚才说的是什么"
  → ChatService.chat(userId, "刚才说的是什么")
    → contextStore.getHistory(userId, 20)  // 上两轮
    → llmClient.chat(userId, "刚才说的是什么", history)
    → LLM 收到完整历史，能理解"刚才"
```

### 3.2 图片分析写入上下文

```
用户发图片
  → VisionHandler
    → downloadImageAsDataUrl()  下载原图
    → visionService.analyze()   分析图片
    → contextStore.append(userId, "user", "[用户发送了一张图片]")
    → contextStore.append(userId, "assistant", "这是一张...")
  → 用户后续问"刚才那张图里的颜色是什么？" → LLM 有上下文能回答
```

### 3.3 引用消息

```
用户引用 "今天天气怎么样" → 打字 "用英文回答"
  → extractRefMessage() 提取 refMessageText = "今天天气怎么样"
  → AIChatHandler.buildFullText()
    → 拼成: [用户引用了: "今天天气怎么样"]\n用英文回答
  → LLM 收到完整上下文

用户引用 bot 发的图片 → 打字 "把背景改成夜晚"
  → extractRefMessage() 提取 refImageEncryptParam / refImageAesKey
  → 后续可用来下载原图进行编辑
```

### 3.4 图文合并

```
用户选图片 + 配文字 "这是什么" 一起发送
  → SDK: itemList = [IMAGE, TEXT]
  → pollLoop: IMAGE+TEXT 合并为一条
    → type=TEXT, text="这是什么", hasAttachedImage=true
    → imageEncryptParam/aesKey 也有值
  → MessageRouter: hasAttachedImage=true → 走意图分类
```
