## 一、已有基础设施（不需要改）

### 1.1 收语音：数据自动填充

用户发语音 → `WechatMessageService` 自动构建 `WechatMessage`：

```java
// pollLoop 已处理 VOICE 类型（MessageItem type=3）
message.getType()                    // MessageType.VOICE
message.getVoiceEncryptQueryParam()  // CDN 下载参数
message.getVoiceAesKey()             // 解密密钥
message.getVoiceText()               // 微信自带转写（可能为 null）
message.getUserId()                  // 用户 ID
```

### 1.2 下载语音文件

```java
// 和图片下载一样
byte[] voiceBytes = wechatClient.downloadMedia(
    message.getVoiceEncryptQueryParam(),
    message.getVoiceAesKey()
);
// 返回 SILK 格式 → 需 ffmpeg 转码
```

### 1.3 发语音

```java
wechatClient.sendVoiceMessage(
    toUserId,         // 用户 ID
    voiceBytes,       // 音频字节（需转 SILK）
    "voice.silk",     // 文件名
    playTimeMs,       // 时长 ms
    6                 // encodeType: 6=SILK
);
```

### 1.4 上下文存储

```java
// 转写文字存对话历史（ChatService 自动处理，不需要手动调）
chatService.chat(userId, transcribedText);

// MP3 路径 + 文字存语音上下文
contextStore.setLastVoice(userId, mp3Path, transcribedText);

// 取上次语音
String[] last = contextStore.getLastVoice(userId);
// last[0] = mp3Path, last[1] = text
```

### 1.5 图片参考

图片的存储模式完全一样，可参考 `VisionHandler` 和 `ImageGenerationHandler`。

## 三、VoiceHandler 流程

```
收到语音消息
  ↓
① 优先用微信自带转写
  if (voiceText != null) → 直接用
  ↓
② 没转写 → 下载 → 转 WAV → ASR
  downloadMedia() → ffmpeg silk→wav → SpeechClient.recognize()
  ↓
③ 存 MP3（转码 SILK→MP3）
  ffmpeg silk→mp3 → 存文件/OSS → contextStore.setLastVoice(userId, path, text)
  ↓
④ 文字走对话流程（自动带上下文）
  ChatService.chat(userId, text) 或 MessageRouter.route(构造TEXT消息)
```

```java
@Component
public class VoiceHandler implements MessageHandler {

    private final WechatILinkClient wechatClient;
    private final ChatService chatService;
    private final ContextStore contextStore;
    private final SpeechClient speechClient;  // 你的 ASR

    @Override
    public WechatReply handle(WechatMessage message) {
        if (message.getType() != MessageType.VOICE) return null;

        // ① 微信自带转写
        String text = message.getVoiceText();
        if (text != null && !text.isEmpty()) {
            return replyWithText(message.getUserId(), text, null);
        }

        // ② 下载语音
        byte[] silk = wechatClient.downloadMedia(
            message.getVoiceEncryptQueryParam(),
            message.getVoiceAesKey()
        );
        if (silk == null) return WechatReply.text("抱歉，语音下载失败");

        // ③ 转 WAV → ASR
        byte[] wav = ffmpegToWav(silk);
        text = speechClient.recognize(wav);
        if (text == null) return WechatReply.text("抱歉，没听清");

        // ④ 存 MP3
        byte[] mp3 = ffmpegToMp3(silk);
        String mp3Path = saveFile(message.getUserId(), mp3);
        contextStore.setLastVoice(message.getUserId(), mp3Path, text);

        return replyWithText(message.getUserId(), text, mp3Path);
    }
}
```

## 四、TTS 发语音

```java
// 1. LLM 返回文字
String reply = chatService.chat(userId, text);

// 2. TTS 生成 MP3
byte[] mp3 = ttsClient.synthesize(reply);

// 3. MP3 → SILK
byte[] silk = ffmpegToSilk(mp3);

// 4. 发送
wechatClient.sendVoiceMessage(userId, silk, "voice.silk", playTimeMs, 6);

// 5. 存上下文（可选，记录 bot 发的语音）
contextStore.setLastVoice(userId, saveFile(mp3), reply);
```

## 六、格式转换参考

```bash
# SILK → WAV（ASR）
ffmpeg -f silk -i input.silk -ar 16000 -ac 1 output.wav

# SILK → MP3（存储）
ffmpeg -f silk -i input.silk output.mp3

# MP3 → SILK（发送）
ffmpeg -i input.mp3 -acodec silk output.silk
```

Java 调 ffmpeg：
```java
ProcessBuilder pb = new ProcessBuilder("ffmpeg", "-f", "silk", "-i", "pipe:0", ...);
Process p = pb.start();
p.getOutputStream().write(silkBytes);
byte[] result = p.getInputStream().readAllBytes();
```
